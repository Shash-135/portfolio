# Shashank Talekar | Full-Stack Developer Portfolio

This portfolio is now a React app built with Vite so it can be deployed to both Vercel and GitHub Pages.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run build
npm run deploy
```

Then set GitHub Pages to serve from the `gh-pages` branch.

## Deploy to Vercel

Import the repository into Vercel and use the default Vite settings. The production build output is `dist`.
